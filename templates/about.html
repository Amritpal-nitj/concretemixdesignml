Project Overview

AI & ML Integrated Concrete Mix Design Predictor (SmartBuild) is an engineering tool that uses Machine Learning to estimate concrete mix proportions consistent with IS 10262:2019 guidelines. The ML model accelerates mix design by quickly suggesting cementitious content, water, fine and coarse aggregate quantities for a required 28-day strength and other site parameters.

Standards & Methodology

  • Standard: IS 10262:2019 ‚Äî Recommended guidelines for concrete mix proportioning.
  • ML Model: Random Forest (multi-output) trained on an IS-derived dataset covering various cement types (OPC/PPC/PSC) and aggregate conditions.
  • Key automated steps: Auto-selection of water-cement ratio (w/c) using IS curves, Factor X, and standard deviation S for target mean strength calculation.

Tech Stack

  • Backend: Flask (Python)
  • ML: scikit-learn (RandomForestRegressor)
  • PDF generation: reportlab
  • Frontend: HTML, CSS (assets/style.css)
